**Smart Dodge Pro – AI-Based Adaptive Difficulty Game**

## Overview

**Smart Dodge Pro** is a 2D interactive game developed using Python and Pygame.
The game demonstrates the concept of **Artificial Intelligence (AI)** through adaptive difficulty, where the system dynamically adjusts the challenge level based on the player's performance.



##  Features

*  Interactive gameplay with smooth controls
*  AI-based adaptive difficulty system
* Real-time score tracking
*  High score system
*  Restart option after game over
*  Clean and simple user interface


##  AI Concept Used

This project uses a **Rule-Based Adaptive AI System**:

* The game monitors the player's score
* Based on performance, the speed of obstacles increases
* This creates a dynamic and engaging gameplay experience


##  Tech Stack

* **Python** – Core programming language
* **Pygame** – Game development library

##  How to Play

* Press **SPACE** → Start the game
* Use ** Left Arrow** → Move left **
* Use ** Right Arrow** → Move right **
* Avoid the falling red blocks
* Each successful dodge increases your score

### Game Controls:

* **R** → Restart after game over
* **Q** → Quit the game


##  Objective

Survive as long as possible by avoiding obstacles and achieving the highest score.


##  How AI Works in This Game

The game uses a simple AI logic:

* Score increases → Game becomes faster
* Higher difficulty → More challenge

This simulates **adaptive intelligence**, a key concept in AI systems.


##  Sample Gameplay Logic

* Score 0–5 → Easy level
* Score 5–10 → Medium level
* Score 10+ → Hard level
